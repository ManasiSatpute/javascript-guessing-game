#  Number Guessing Game (JavaScript)

This is a simple **Number Guessing Game** built using **Vanilla JavaScript**.  
The game generates a random number and asks the user to guess it until the correct number is found or the user quits.

---

##  Features
- Generates a random number using `Math.random()` and `Math.floor()`
- User can set the maximum number range
- Provides hints if the guess is too high or too low
- Option to quit the game anytime
- Beginner-friendly JavaScript project

---

##  Technologies Used
- JavaScript (Vanilla JS)
- Browser `prompt()` and `console.log()`
